//查看本地仓库的情况，命令用于显示工作目录和暂存区的状态。使用此命令能看到那些修改被暂存到了,哪些没有。（当前红色的区域就是你本地修改的）
1.git status 
//添加单个文件到提交列表
2.git add .
//查看一下 若文件变成绿色则这些文件可以被提交到远程仓库中去（这个命令不能完成提交只是存到了暂存区中）
3.git status 
 //你提交的代码是实现什么功能在“”中体现
4.git commit -m “xxxxxxxx”
//代码提交后执行此命令则可以看到你的提交，最新的commit就是你提交的，commit后有commit号，这个号是唯一的。
5.git log 
//可以看你写的代码
6.git show commit号 
到这一步为止就只是把代码更新到了本地仓库，还没推送到远端的仓库
//将你当前的分支名推送到远端（你现在的分支名是什么推送到远端的分支名就是什么）
7.git push origin 分支名 
//查看远程版本库的分支列表
8.git branch -r
